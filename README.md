# Scalehero

A heroku auto-scaler using heroku-api

## Installation

Add this line to your application's Gemfile:

    gem 'scalehero'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install scalehero

## Usage

Just make sure that you have these ENV variables:

    ENV['HEROKU_APP']

and

    ENV['HEROKU_API_KEY']
